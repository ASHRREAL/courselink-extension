# CourseLink+ Privacy Policy

Last updated: April 15, 2026

CourseLink+ does not collect, store, or transmit any user data. All features run locally in your browser. Settings are saved using Chrome's local storage and are never sent to any server.

The extension only modifies pages on courselink.uoguelph.ca and does not track, profile, or share any information.
